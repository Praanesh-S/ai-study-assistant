# 🧠 AI Study Assistant

> An AI-powered study companion that summarizes notes, generates quizzes, and creates flashcards — built with Gemini AI.

🔗 **Live Demo:** (https://ai-study-assistant-bypraanesh.streamlit.app)

---

## ✨ Features

- 📝 **Summarize** — paste notes or upload a PDF and get a clean, concise summary
- ❓ **Quiz Generator** — auto-generate 5 quiz questions with answers from your material
- 🃏 **Flashcards** — generate question/answer flashcard pairs for quick revision
- 📄 **PDF Upload** — extract and process text directly from PDF files
- ⚡ **Powered by Gemini 2.0 Flash** — fast and accurate AI responses

---

## 🛠️ Tech Stack

| Layer | Tool |
|-------|------|
| Frontend | Streamlit |
| AI Model | Google Gemini 2.0 Flash |
| PDF Processing | PyMuPDF |
| Deployment | Streamlit Cloud |
